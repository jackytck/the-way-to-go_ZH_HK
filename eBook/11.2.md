# 11.2 介面巢狀介面

一個介面可以包含一個或多個其他的介面，這相當於直接將這些內嵌介面的方法列舉在外層介面中一樣。

比如介面 `File` 包含了 `ReadWrite` 和 `Lock` 的所有方法，它還額外有一個 `Close()` 方法。

```go
type ReadWrite interface {
    Read(b Buffer) bool
    Write(b Buffer) bool
}

type Lock interface {
    Lock()
    Unlock()
}

type File interface {
    ReadWrite
    Lock
    Close()
}
```
